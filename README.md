## Overview

Develop event-driven trading signals based on microstructure market events such as:

- Volume spikes
- Spread widening
- Queue depletion at best bid/ask

Analyze **post-event price dynamics**, hit rates, and adverse selection effects. Designed as a portfolio project to demonstrate:

- Quantitative research skills
- Event-based trading strategy design
- Python data analysis and visualization
